# brand-feathercat-products

**Brand:** FeatherCat  
**Domain:** https://feathercat.com (apex) and https://www.feathercat.com  
**Category:** Ultra-light modular catamarans — river & ocean, paddle/row (and later sail)

## Purpose
This repository hosts the public-facing product definitions, documentation, and marketing-ready assets for **FeatherCat**.

FeatherCat focuses on **small-to-medium ultra-light catamarans** designed around modular transport and serviceability:
- river-capable configurations (lightweight, durable, easy to field service)
- ocean-capable configurations (stability and efficiency with saltwater-ready accessories)
- primary propulsion: paddling or rowing
- future path: optional sailing configurations

## Commerce & Memberships
Sales, memberships, and checkout will be handled by **WaterWolf** (private platform).  
This repo is intentionally **public** and contains only publishable product and documentation assets.
